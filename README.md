# Video Shelf

* A full-stack CRUD app for keeping a database of the movies in your personal collection.

* Click [here](https://video-shelf-app.herokuapp.com/) to view the app.

### Technology Implemented: 

* JavaScript
* Node | Express
* HTML | EJS
* CSS
* Heroku
* GitHub
